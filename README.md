## 🚀 Installation

To get this project up and running locally, follow these steps. Make sure you have Node.js and npm (or yarn) installed on your system.

### Prerequisites

*   Node.js (LTS version recommended)
*   npm or Yarn
*   Gatsby CLI: If you don't have it, install it globally:
    ```bash
    npm install -g gatsby-cli
    # or
    yarn global add gatsby-cli
    ```

### Steps

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Dev-moe-kyawaung/supreme-quantumleaf-gatsby.git
    cd supreme-quantumleaf-gatsby
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Start the development server**:
    ```bash
    gatsby develop
    ```

    This command will start a development server at `http://localhost:8000`. You can now open your browser and navigate to this URL to see the portfolio.

## 💡 Usage

Once the development server is running, you can interact with the portfolio locally. This project is intended as a personal portfolio, showcasing the author's skills and projects. Visitors can browse through sections to learn about the 'Digital Architect • Systems Thinker'.
